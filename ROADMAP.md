# Roadmap

Planned improvements and future evolution of the boilerplate.

## v1.x (Stability & Improvements)

* Tooling hardening and cleanup
* Further PostCSS simplification
* Clearer dev / production separation
* Performance optimizations
* Improved documentation and DX
* Minor architectural refinements without breaking changes

---

## v2.0.0 (Major Evolution – Exploratory)

* Composer integration for WordPress dependencies
* Possible migration to Vite (if justified by project needs)
* Improved module encapsulation
* Potential separation between workspace and theme package
* CI/CD pipeline preparation
* Optional full TypeScript adoption
* Strict typing configuration
* Potential introduction of ESLint + type-aware rules

### Infrastructure evolution

- Evaluate Docker / DDEV as a replacement for MAMP
- Standardize development environment across projects
- Improve reproducibility and onboarding

---

This roadmap is indicative and may evolve depending on internal needs and technical constraints.
